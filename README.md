# Dataix3 Website Concept

A professional, meaningful and responsive concept website for Dataix3, showcasing services in Web, Data, Cloud, and Marketing.

## Features

- **Dynamic Hero Section**: Interactive background slider with glassmorphism effects.
- **Comprehensive Services**: Tabbed interface covering Web, Data, Cloud, and Marketing (including Brand Outreach & Portfolio Making).
- **Modern UI**: utilizes "Bento Grid" layout for feature highlights.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Animations**: Smooth transitions powered by Framer Motion.

## Technologies Used

- **Framework**: React + Vite
- **Styling**: CSS Modules / Vanilla CSS
- **Animations**: Framer Motion
- **Icons**: React Icons

## Getting Started

Follow these instructions to get the project up and running on your local machine.

### Prerequisites

Ensure you have Node.js installed on your system.

### Cloning the Repository

```bash
git clone https://github.com/Amaljithuk/dataix3.git
cd dataix3
```

### Installation

Install the project dependencies:

```bash
npm install
```

### Running the App

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or the port shown in your terminal).

### Building for Production

To create a production-ready build:

```bash
npm run build
```

## Structure

- `src/components`: Contains all UI sections (Hero, Services, Features, etc.)
- `src/App.jsx`: Main application layout.
